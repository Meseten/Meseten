### Projects

* **[PASADA](https://github.com/Meseten/PASADA)** – A cross-platform desktop application engineered to streamline manual data entry and document processing for government operations. It features automated field-filling logic (e.g., dynamically generating the "Date Issued" and "Valid Until" fields when an SBN is modified). Built primarily in Python, it leverages Cython, C, and C++ for maximum execution speed, and relies on UDP/HTTP protocols for seamless local network synchronization.
* **[Void Strike - Java Edition](https://github.com/CirJ9/ALFARERO-BAGUISI-BRAVO-DUAG-GARCIA-BSCS1B)** – A 2D arcade shooter inspired by Void Strike, built entirely from scratch using Java and Apache NetBeans. For this project, I programmed the core gameplay mechanics and logic, designed the user interface, and integrated all custom visual assets, sprite animations, and sound effects.
* **[ben4dev](https://github.com/Meseten/ben-dev)** – A highly optimized web portfolio built with Next.js 14, TypeScript, Tailwind CSS, and Framer Motion. It simulates a fast, interactive Linux Kernel boot sequence right in the browser.
* **[AGWA Water Services, Inc.](https://github.com/Meseten/AGWA-Water-Services-Inc.-DPR)** – An algorithmic utility management system developed for a local water district. The platform actively integrates regression forecasting models alongside a robust digital payment infrastructure.
* **[SchedUltimaPro](https://github.com/Meseten/cpu-scheduler)** – A dedicated TypeScript implementation detailing operating system CPU scheduling algorithms, calculating process execution times and performance metrics.
* **[NextQue3.0](https://github.com/Meseten/NextQue3.0)** – A structural Java-templated queue management application designed and optimized for efficient data processing flows.
